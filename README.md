# Leenalchi Cafe — Telegram-бот системи лояльності

Бот замінює сайт + SMS-підтвердження. Клієнт підтверджує номер телефону
натисканням кнопки в Telegram (це робить сам Telegram, безкоштовно і
без OTP-кодів). Адмін нараховує/списує бали прямо в чаті з ботом.

## 1. Отримати токен бота

1. Відкрий Telegram, знайди **@BotFather**
2. Напиши `/newbot`, дай боту ім'я та юзернейм (напр. `leenalchi_cafe_bot`)
3. BotFather видасть токен виду `123456789:AAExampleTokenXXXXXXXXXXXXXXXXXXX`

## 2. Дізнатись свій Telegram ID (щоб стати адміном)

Напиши в Telegram боту **@userinfobot** — він покаже твій числовий ID.
Це ID потрібно вписати в `ADMIN_IDS`.

## 3. Запуск локально (для тесту)

```bash
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt

export BOT_TOKEN="токен_від_BotFather"
export ADMIN_IDS="твій_telegram_id"      # через кому, якщо адмінів декілька

python bot.py
```

Бот почне відповідати одразу після запуску. Зупиняється — Ctrl+C.

## 4. Команди

**Клієнт**
- `/start` — реєстрація (кнопка "Поділитися номером")
- `/balance` — баланс і прогрес до безкоштовного напою
- `/history` — останні операції

**Адмін** (тільки для ID зі списку `ADMIN_IDS`)
- `/admin` — довідка
- `/find +380XXXXXXXXX` — знайти клієнта
- `/add +380XXXXXXXXX 50 замовлення №42` — нарахувати бали
- `/sub +380XXXXXXXXX 20` — списати бали
- `/register +380XXXXXXXXX Ім'я` — зареєструвати клієнта вручну (наприклад, якщо він ще не запускав бота, а бариста хоче видати йому картку лояльності на касі)

## 5. Постійна робота (24/7)

Локальний запуск працює, поки відкритий термінал. Для продакшена бот має
працювати постійно на сервері. Найпростіші варіанти для старту:

- **Railway.app** або **Render.com** — безкоштовний/дешевий "Background
  Worker", просто підключаєш GitHub-репозиторій з цим кодом, задаєш
  `BOT_TOKEN` і `ADMIN_IDS` як Environment Variables — і все запускається.
- **Звичайний VPS** (наприклад, DigitalOcean, Hetzner ~$4-5/міс) +
  `systemd`-сервіс або `tmux`/`screen`, щоб процес не зупинявся після
  виходу з SSH.
- **PythonAnywhere** — має безкоштовний тариф, підходить для дуже
  маленького навантаження.

## 6. База даних

Дані зберігаються у файлі `leenalchi.db` (SQLite) поруч з `bot.py`.
Для старту кав'ярні цього більш ніж достатньо. Якщо згодом знадобиться
кілька адмінів, що працюють одночасно з великим потоком клієнтів —
можна перенести на PostgreSQL без зміни логіки бота.

## Чому бот дешевший за сайт+SMS

- Telegram підтверджує номер телефону клієнта сам, безкоштовно —
  не потрібен платний SMS-провайдер (TurboSMS/Twilio) для OTP.
- Не потрібен хостинг фронтенду й бекенду окремо — один процес.
- Клієнту не треба нічого встановлювати — Telegram майже у всіх вже є.
